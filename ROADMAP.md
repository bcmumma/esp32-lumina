# Lumina Roadmap

## v1.0 — Working Prototype

Status: Complete

- End-to-end PCM1808 to ESP32-S3 to ESP-NOW to LED receiver pipeline
- Browser UI
- Multi-strand LED support
- Color calibration
- Recording support

## v2.0 — Project Architecture

- Migrate to PlatformIO
- Split firmware into modules
- Create sender and receiver architecture
- Improve repository structure
- Add diagnostics page
- Clean up web UI
- Improve settings management

## v2.1 — Lighting Engine

- Add effect library
- Add scene system
- Add palettes
- Add transitions
- Add better multi-zone support

## v2.2 — Audio Engine

- Improve FFT engine
- Improve beat detection
- Add BPM estimation
- Add beat confidence
- Add energy tracking
- Add adaptive sensitivity

## v3.0 — Smart Home Integration

- Add MQTT
- Add Home Assistant discovery
- Expose lights and sensors
- Add automation hooks

## v4.0 — AI Scene Generation

- Add local LLM scene generation
- Convert natural language prompts into lighting scenes
- Add scene presets
- Add AI-assisted palette generation
