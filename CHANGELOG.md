# Changelog

## v1.0.0

Initial working prototype release.

### Added

- PCM1808 I2S audio input
- ESP32-S3 sender firmware
- ESP32-WROOM receiver firmware
- ESP-NOW packet communication
- FFT-based audio analysis
- Beat detection
- Browser-based control UI
- LED Colors page
- Hardware configuration page
- Audio tuning page
- Recorder support
- Multi-strand LED support
- WS2815 / WS2812-style LED support
- Color calibration
- Saveable settings
- Python recorder helper
- Audio test WAV generator

### Known Limitations

- Firmware is not yet modularized.
- Beat detection needs further refinement.
- Wi-Fi channel matching is manual.
- No Home Assistant integration yet.
- No MQTT support yet.
- No AI scene generation yet.
